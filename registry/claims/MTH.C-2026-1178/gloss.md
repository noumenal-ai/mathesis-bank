Statement of record: `Eidometry.Thesis.extension_eq_forced_fiber` (Eidometry thesis, section Non-triviality).

The set of global states that are compatible with all local observations across a cover is built with no reference to any quotient or partition; the theorem proves that once the cover reaches every observable, this set exactly equals the set of states that agree with a given state on every observation. The two objects coincide by proof, not by definition.
