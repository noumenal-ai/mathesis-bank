Statement of record: `Eidometry.Thesis.covering_dichotomy` (Eidometry thesis, section Non-triviality).

A cover lets every pairwise-consistent, locally-nonempty family of local relations glue into a single global object exactly when the cover is acyclic. This equivalence is proved in a separate library (a discrete version of Vorob'ev's gluing theorem), and it is what separates the covers where local data determines the global object from those where it cannot.
